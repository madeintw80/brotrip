## 📱 第一次怎麼開始？

### iPhone（Safari）

1. 打開 **Safari**（不要用 LINE 內建瀏覽器！會看到 OAuth 報錯）
2. 網址列輸入：`https://madeintw80.github.io/brotrip/`
3. 「用 Google 登入」→ **選 Wei 加你權限那個 Gmail**
4. 自動找到群組進主畫面 ✅
5. 想變 App：Safari 下方分享鍵 → 「加入主畫面」

### Android（Chrome）

1. Chrome 開：`https://madeintw80.github.io/brotrip/`
2. Google 登入 → 同上
3. 變 App：Chrome 右上選單 → 「安裝應用程式」

### ⚠️ 為什麼不能在 LINE 內登入？

Google 限制：app 內建瀏覽器（LINE/FB/IG/Messenger）無法做 OAuth，會看到「origin parameter is required」400 錯誤。**必須用 Safari/Chrome 外部瀏覽器**。

## 🗂 5 個 Tab 怎麼用

底下從左到右：💰 記帳 / 📸 日記 / 🗺 地圖 / 💡 願望 / ⚙ 設定

### 💰 記帳
- 右下角 ➕ → 加新支出
- 選誰付的、誰要分（可一人付多人分）
- 自動算「誰欠誰多少」（💵 結算）
- 結清後對方按「✅ 確認收到」雙方都安心

### 📸 日記
- 右下角 ➕ → 寫日記（可加多張照片）
- 文字內打 `@` 可以 tag 朋友（會推播給他）
- **小技巧**：地點欄位下方有「💡 從願望清單選」，點一下自動帶地點，存日記後該 wish 自動標已去過 ✓

### 🗺 地圖
- 看大家的日記+行程地點
- **三色 marker**：
  - 🟡 黃色（半透明）= 願望清單想去
  - 🟢 綠色 = 已排進行程
  - ⚪ 灰色 = 已去過
- 點 marker 可以「✓ 標已去過」或「➕ 加進行程」或「🗺️ 開 Google Maps」
- 上方 ☑ 可關掉 wishlist 地點顯示

### 💡 願望（這個最有趣）
出國前**大家把想去的地方都丟進來**，現場再決定。

**怎麼加**：
- 右下角 ➕ → 輸入地名（會跳 Google 推薦）→ 選類型（餐廳/景點/咖啡/...）→ 寫備註（誰推薦的 / 為什麼想去）

**怎麼用**：
- 🍜 餐廳 / 🏛️ 景點 / ☕ 咖啡 chips：按分類篩選
- 👥 全部 / Paul / Kate ... chips：看誰加的
- **排序**：⏰ 最近加的 / **📍 距離我最近** / 🔤 名稱
  - 「距離我最近」會自動分組：「500m 內」/「500m–2km」/「2km 以上」→ 走路上看「現在能去哪些」
- **👎 否決票**：對某人加的 wish 按 👎，超過半數自動隱藏（避免大家不想去的硬塞）
- **➕ 加進行程**：選「建新行程」或「加進既有行程」
- **🗺️ 地圖**：跳 Google Maps 看詳細
- **✏️ 改備註**：自己加的可以隨時改備註（推薦來源、為什麼想去）

### ⚙ 設定
- 切換 dark / light mode
- 🔄 **檢查新版本**（每次 Wei 更新後按一次就好，不用滑掉重開）
- 🔔 開啟附近 wish 推播（走 500m 內自動提醒）
- 群組管理（建群組 / 加入 / 退出）

## 🔔 附近 wish 推播（出國神器）

進「💡 願望」tab 會跳 onboarding 問「要不要開附近推播」。

開了之後：
- App 開著 → 走在路上經過 wish 500m 內 → 跳通知「附近有想去的『XXX』」
- App 沒在用但**沒從 App Switcher 滑掉** → 仍會收到系統通知
- App 完全 kill (滑掉) → 收不到（iOS PWA 限制）

不想接收？設定 tab 可以隨時關。

## 🆘 救急 SOP（最重要！這段救過很多次）

照順序試，**任一步驟成功就停**。

### 1️⃣ 卡在「找不到群組」？

進得了 BroTrip 主畫面但看不到群組？
- 設定 → 「🔄 從 Drive 重新偵測群組」按下去
- 或無群組畫面的綠色大按鈕「🔄 重新搜尋我的群組」

### 2️⃣ BroTrip 一直要求重登 / 看到無群組畫面？

→ 多半是**登入用了不對的 Gmail**！

- 設定 → Debug 區 → 看「Email」是哪個 Gmail
- 把這個 Gmail 截圖給 Wei，他確認 ACL 內是不是這個 email
- 如果不是 → ⏏ 登出 → 重登時選正確 Gmail
- 或 v3.8.1+ 用「📨 用邀請碼加入」會跳出「🔄 換帳號重登」紅色按鈕，一鍵解決

### 3️⃣ 邀請流程跳 Drive 一直要存取權？

→ 你 BroTrip 登入用的 Gmail 跟 Wei 加你權限的 Gmail 不一樣！

- 看 banner 上「你現在用 XX@gmail.com」
- 是不是 Wei 加你權限那個？不是 → 按紅色「🔄 換帳號重登」
- 你不確定是哪個？問 Wei

### 4️⃣ 卡在登入畫面 / 無群組畫面，按不到設定？

v3.8.2 開始：**這兩個畫面左下角**都有版本號 + 「🔄 強制更新」小按鈕，按一次強制升級。

如果連這個按鈕都看不到（你版本太舊）：

**用 Safari 直接開新版**（最暴力但 100% 解）：
1. 完全 kill 桌面 PWA
2. Safari 開 `https://madeintw80.github.io/brotrip/`
3. 拿到最新版（Safari 沒 SW cache 限制）
4. 登入 → 正常進去後再「加入主畫面」變新版 PWA

### 5️⃣ Workers 後端掛了 / 邀請碼不能用？

- 設定 → Debug 區「Workers 狀態」應該顯示 ✅ 正常
- 如果 ❌ 連線失敗 → 告訴 Wei，可能是他 Cloudflare account 異常

### 6️⃣ 終極殺手（其他都不行才用）

⚙ 設定 → 🚨 **「重置」**紅色按鈕 → 清 SW + 登出 + 重來
- 不會清掉你的暱稱/日記/支出資料（那些在 Sheet）
- 只清本機 PWA 殼

## ❓ 其他常見問題

### Q：剛剛新增的東西沒出現？
A：下拉刷新（手指從上往下拉）。或設定 → 「🔄 檢查新版本」。

### Q：要換手機怎麼辦？
A：新手機 Safari 開 https://madeintw80.github.io/brotrip/ → Google 登入 → 自動偵測你的群組（資料在 Drive 不會丟）。

### Q：朋友資料會看到我的 Drive 嗎？
A：不會。BroTrip 只能看到自己跟群組共享的 Sheet/相簿，看不到你其他 Drive 檔案。

### Q：花費怎麼結算？
A：💰 記帳 tab 上方有「💵 結算」區，會自動算「A 欠 B 多少」。轉帳給對方後點「我已轉帳」，對方點「✅ 確認收到」就結清。

### Q：照片存哪？
A：存在 Wei 的 Google Drive 內（群組共享資料夾）。日記刪了照片也會刪。

### Q：暱稱怎麼改？
A：設定 → 群組管理 → 「👤 我在此群組的顯示名稱」。

### Q：邀請碼那串好長好嚇人？
A：v3.6.1+ 已縮到 **8 個字**（透過 Cloudflare backend），不再是長字串。如果你看到的還是長字串，按「🔄 強制更新」拿新版。

### Q：用 LINE 點 Wei 給的連結卡住？
A：LINE 內建瀏覽器不能登 Google。長按連結 → 「拷貝」→ 開 Safari → 貼網址 → 前往。

## 🆘 卡住找誰？

如果以上 SOP 都試過還不行：
- LINE 私訊 Wei，**附上**：
  - BroTrip 設定 tab → Debug 區整段截圖（含 Email / Workers 狀態 / App version）
  - 你卡住的畫面截圖
  - 簡述哪一步卡住

Wei 那邊可以：
- 看 Sheet 後端資料修
- 用 Drive API 直接幫你加權限
- 升級 / 重設你的群組
- 加新功能
