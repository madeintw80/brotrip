# 🎒 BroTrip 使用攻略（5 男出遊神器）

> 朋友看的簡易版。複製整段貼到 LINE 群也可讀。

---

## 📱 第一次怎麼開始？

1. **點 [Wei 給的邀請連結]** → 自動開啟 BroTrip
2. **用 Google 帳號登入**（任何 Gmail 都行）
3. 跳出邀請畫面 → 按「**加入**」
4. 設定你在群組裡的暱稱（之後可改）
5. 完成！可以開始記帳寫日記了 🎉

### 💡 強烈建議「加到主畫面」（變成 App 一樣）
- **iPhone**：Safari 開 BroTrip → 點下方分享鍵 → 「**加入主畫面**」
- **Android**：Chrome 開 → 右上選單 → 「**安裝應用程式**」
- 加完後從主畫面點圖示開啟，跟一般 App 一樣

⚠️ 第一次從主畫面開啟會要你**重登一次 Google**（這是 iOS PWA 限制），登完會自動同步群組，之後就不用再登。

## 🗂 5 個 Tab 怎麼用

底下從左到右：💰 記帳 / 📸 日記 / 🗺 地圖 / 💡 願望 / ⚙ 設定

### 💰 記帳
- 右下角 ➕ → 加新支出
- 選誰付的、誰要分（可一人付多人分）
- 自動算「誰欠誰多少」（💵 結算）
- 結清後對方按「✅ 確認收到」雙方都安心

### 📸 日記
- 右下角 ➕ → 寫日記（可加多張照片）
- 文字內打 `@` 可以 tag 朋友（會推播給他）
- **小技巧**：地點欄位下方有「💡 從願望清單選」，點一下自動帶地點，存日記後該 wish 自動標已去過 ✓

### 🗺 地圖
- 看大家的日記+行程地點
- **三色 marker**：
  - 🟡 黃色（半透明）= 願望清單想去
  - 🟢 綠色 = 已排進行程
  - ⚪ 灰色 = 已去過
- 點 marker 可以「✓ 標已去過」或「➕ 加進行程」或「🗺️ 開 Google Maps」
- 上方 ☑ 可關掉 wishlist 地點顯示

### 💡 願望（這個最有趣）
出國前**大家把想去的地方都丟進來**，現場再決定。

**怎麼加**：
- 右下角 ➕ → 輸入地名（會跳 Google 推薦）→ 選類型（餐廳/景點/咖啡/...）→ 寫備註（誰推薦的 / 為什麼想去）

**怎麼用**：
- 🍜 餐廳 / 🏛️ 景點 / ☕ 咖啡 chips：按分類篩選
- 👥 全部 / Paul / Kate ... chips：看誰加的
- **排序**：⏰ 最近加的 / **📍 距離我最近** / 🔤 名稱
  - 「距離我最近」會自動分組：「500m 內」/「500m–2km」/「2km 以上」→ 走路上看「現在能去哪些」
- **👎 否決票**：對某人加的 wish 按 👎，超過半數自動隱藏（避免大家不想去的硬塞）
- **➕ 加進行程**：選「建新行程」或「加進既有行程」
- **🗺️ 地圖**：跳 Google Maps 看詳細

### ⚙ 設定
- 切換 dark / light mode
- 🔄 **檢查新版本**（每次 Wei 更新後按一次就好，不用滑掉重開）
- 🔔 開啟附近 wish 推播（走 500m 內自動提醒）
- 群組管理（建群組 / 加入 / 退出）

## 🔔 附近 wish 推播（出國神器）

進「💡 願望」tab 會跳 onboarding 問「要不要開附近推播」。

開了之後：
- App 開著 → 走在路上經過 wish 500m 內 → 跳通知「附近有想去的『XXX』」
- App 沒在用但**沒從 App Switcher 滑掉** → 仍會收到系統通知
- App 完全 kill (滑掉) → 收不到（iOS PWA 限制）

不想接收？設定 tab 可以隨時關。

## ❓ 常見問題

### Q：剛剛新增的東西沒出現？
A：下拉刷新（手指從上往下拉）。或設定 → 「🔄 檢查新版本」。

### Q：群組不見了？
A：設定 → 「🔄 從 Drive 重新偵測群組」會自動找回所有你被邀請加入的群組。

### Q：登出了 / 不小心換了 Google 帳號？
A：用原本邀請連結再點一次，重登正確的 Google 帳號就會自動加回群組。

### Q：要換手機怎麼辦？
A：新手機開瀏覽器到 BroTrip → Google 登入 → 自動偵測你的群組（資料在 Drive 不會丟）。

### Q：朋友資料會看到我的 Drive 嗎？
A：不會。BroTrip 只能看到自己跟群組共享的 Sheet/相簿，看不到你其他 Drive 檔案。

### Q：花費怎麼結算？
A：💰 記帳 tab 上方有「💵 結算」區，會自動算「A 欠 B 多少」。轉帳給對方後點「我已轉帳」，對方點「✅ 確認收到」就結清。

### Q：照片存哪？
A：存在 Wei 的 Google Drive 內（群組共享資料夾）。日記刪了照片也會刪。

### Q：暱稱怎麼改？
A：設定 → 群組管理 → 「👤 我在此群組的顯示名稱」。
